# 🎓 Virtual Courses - AI Powered Learning Management System

An AI-powered Learning Management System (LMS) built with the MERN Stack that enables students to discover relevant courses, manage their learning journey, and interact with educators through a modern and responsive platform.

## 🚀 Features

### 🔐 Authentication & Security
- JWT Authentication
- Google OAuth Login
- Protected Routes
- Role-Based Access Control (Student / Educator)

### 🤖 AI-Powered Course Discovery
- AI-based course search
- Personalized course recommendations
- Smart learning suggestions

### 👤 User Profile Management
- Update profile information
- Upload profile pictures
- Cloudinary image storage
- Secure account management

### 📚 Course Management
- Browse available courses
- View detailed course information
- Educator course management
- Student learning dashboard

### 🎨 Modern UI/UX
- Fully Responsive Design
- Mobile Friendly
- Tailwind CSS Styling
- Interactive Components

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/virtual-courses.git
cd virtual-courses
```

### Backend Setup

```bash
cd backend
npm install
npm run server
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder:

```env
MONGO_URI=
JWT_SECRET=

CLOUDINARY_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

EMAIL=
EMAIL_PASSWORD=
```

Create a `.env` file in the frontend folder:

```env
VITE_FIREBASE_API_KEY=
```
